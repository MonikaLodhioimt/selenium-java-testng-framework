# Selenium Java Automation Framework

## Overview
This project is a UI automation testing framework developed using Selenium WebDriver, Java, TestNG, and Cucumber (BDD) following the Page Object Model design pattern.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Cucumber (BDD)
- Maven
- Git

## Features
- Page Object Model (POM) design
- Reusable test components
- Test execution using TestNG
- BDD support with Cucumber
- Easy maintenance and scalability

## Project Structure
src/test/java
- base – Base test setup
- pages – Page classes
- tests – Test cases
- utils – Utility classes

## How to Run
1. Clone the repository
2. Open project in Eclipse/IDE
3. Run testng.xml or use:
   mvn clean test

## Sample Test Scenarios
- Login functionality test
- Invalid login validation
- UI element verification
